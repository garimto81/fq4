# PRD-0001: First Queen 4 Remake

**Version**: 1.0.0 | **Date**: 2026-01-30 | **Status**: Draft

---

## Executive Summary

First Queen 4는 1994년 Kure Software Koubou가 개발한 실시간 전술 RPG입니다. 독창적인 **Gocha-Kyara 시스템**을 통해 대규모 부대 전투를 구현한 게임으로, 현대 플랫폼을 위한 HD 리메이크를 제안합니다.

| 항목 | 내용 |
|------|------|
| **원본** | First Queen 4 (DOS, 1994) |
| **장르** | 실시간 전술 RPG |
| **개발 기간** | 14개월 (Pre-production 2개월 + MVP 6개월 + Full 6개월) |
| **타겟 플랫폼** | PC (Steam), Nintendo Switch |
| **타겟 유저** | 레트로 게이머, 전술 RPG 팬 |

---

## 1. 프로젝트 배경

### 1.1 원본 게임 소개

First Queen 시리즈는 "Gocha-Kyara"(ゴチャキャラ) 시스템으로 유명합니다:

- **실시간 전투**: 턴제가 아닌 실시간 전술 전투
- **AI 동료 제어**: 플레이어는 1명만 직접 제어, 나머지는 AI
- **대규모 부대**: 수십 명의 유닛이 동시에 전투
- **피로도 관리**: 전투 지속력을 관리하는 독특한 시스템

### 1.2 리메이크 필요성

| 이유 | 설명 |
|------|------|
| **접근성** | DOS 게임으로 현대 환경에서 플레이 어려움 |
| **그래픽** | 16색 320x200 해상도는 현대 기준 미달 |
| **QoL** | 세이브/로드, UI/UX 현대화 필요 |
| **시장 기회** | Unicorn Overlord 성공으로 실시간 전술 RPG 수요 증가 |

### 1.3 레퍼런스 게임

| 게임 | 참고 요소 |
|------|----------|
| **Unicorn Overlord** (2024) | 실시간 분대 이동, AI 행동 설정 |
| **Tactics Ogre Reborn** (2022) | HD 리마스터 방식, BGM 재해석 |
| **Live A Live HD-2D** (2022) | HD-2D 그래픽 스타일 |

## 2. 핵심 게임플레이

### 2.1 Gocha-Kyara System

First Queen의 핵심 시스템을 충실히 재현합니다.

```
┌─────────────────────────────────────────────────────────┐
│                  Gocha-Kyara System                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [플레이어]          [AI 동료들]                        │
│   직접 제어 (1명) ←→ 자동 행동 (수십 명)                 │
│                                                         │
│   조작 키:                                              │
│   ← →  : 부대 내 캐릭터 전환                           │
│   ↑ ↓  : 군단(부대) 전환                               │
│   SPACE: 특수 능력/마법 발동                            │
│                                                         │
│   AI 행동 원칙:                                         │
│   - 가까운 적 자동 공격                                 │
│   - 리더 따라가기 (Follow)                              │
│   - 피로도 낮으면 자동 후퇴                             │
│   - 개별 성격에 따른 행동 변화                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 피로도(Fatigue) 시스템

전투 지속력 관리가 핵심 전략입니다.

| FT Level | 효과 | 시각 표시 |
|----------|------|----------|
| 100-70% | 정상 전투력 | 녹색 바 |
| 69-40% | 공격력 -20%, 회피 -10% | 황색 바 |
| 39-20% | 공격력 -50%, 회피 -30% | 적색 바 |
| 19-0% | 자동 후퇴, 전투 불가 | 점멸 |

**회복 방법:**
- 안전 지역 대기 (+3/초)
- 모래시계 아이템 (+50)
- 페어리 샘 (무료 완전 회복)
- 숙영/여관 (전원 회복)

### 2.3 RPG 요소

| 요소 | 설명 |
|------|------|
| **레벨 시스템** | 경험치 획득 → 스탯 성장 |
| **장비** | 무기/방어구 장착, 특수 아이템 |
| **마법** | 최대 5개 슬롯, 상점/아이템으로 습득 |
| **부대 관리** | 병사 모집, 진형 편성 |
